# sc-web
Security Council web interface

## Deployment instructions

```
$ sudo pip install corsa
[$ cd sc-web]
$ corsa --app-dir . --allow-proxy http://52.72.172.54:8080 --allow-origin ALL
```
Then browse to localhost:8888/app/pageName.html
